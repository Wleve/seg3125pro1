About us

This website was created by William Levesque, a Software Engineering student at the university of Ottawa. The goal of this website was to further improve his knowledge on full-stack web development. The website was created using HTML, CSS, and JavaScript. The website is hosted on github pages and uses MongoDB to store all user information. The website is a simple food ordering website where users can create an account, login, and order food.
